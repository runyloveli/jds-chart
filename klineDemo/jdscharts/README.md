
## File
+ `demo` 自己写的一些例子
+ `extension` （扩展中使用）
+ `lib` （源码中没有，执行webpack编译后才存在）
+ `map` （世界地图，中国地图及中国各个省份地图的js和json两种格式的文件）
+ `src` （核心源码）
+ `test` （示例demo）
+ `theme` （主题）

源码入口文件在`src/echarts.js`

## Package
打包命令如下，详见博客说明。
```bash
npm install  //安装所有依赖包
webpack      //打包
webpack -p   //打成压缩包（.min.js）
```
